## 1. 🎓 Student Performance Prediction

**Period:** 11/2025 – 12/2025 &nbsp;|&nbsp; **Grade:** A &nbsp;|&nbsp; **Role:** Team Leader

Xây dựng end-to-end ML pipeline dự đoán điểm thi sinh viên từ 6,600+ bản ghi. So sánh 4 mô hình hồi quy và kiểm định 4 giả định thống kê.

**Results:**
- Cải thiện R² từ **−0.94 → 0.96** qua feature engineering & cross-validation
- Polynomial Regression (degree=2) đạt kết quả tốt nhất: **R² = 0.9609, RMSE = 0.66**
- Kiểm định đầy đủ: Linearity ✅ · Independence (DW≈2) ✅ · Normality ✅

**Tech stack:** `pandas` `numpy` `scikit-learn` `matplotlib` `seaborn` `statsmodels`
